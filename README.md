# Openwebvulndb (Tools)

This project consists of a collection of tools to maintain the vulnerability
databases.


## Set-up

General python set-up:

```
virtualenv -ppython3.5 .
source bin/activate
pip install -r requirements.txt
pip install -r requirements-dev.txt

nosetests
```
